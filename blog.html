Difference Between Session Storage and local Storage scope in JavaScript

in JavaScript local storage and session storage are almost same kind of thing but there are some difference! before we discuss about the difference let's say what are them!
local storage and session storage both are used for store some data on the browser or client side for later usecase, since both are used to store data on the browser but they had some fundamental difference. let see what are the difference.
the data stored in the session storage are cleared whenever the user close that particular tab or window of the browser!

on the other hand the data were stored in the local storage they are never expire! another fundamental difference between then is the capacity of the bandwidth. in local storage we can store up to 10 MB of dada and in session storage we can store only 5 MB of data we can access local storage data from any window but we cannot access the session storage data without the current tab
